# fdp-calc
calculates fdp

HOW TO REFRESH THE APP:

For Chrome on Android
1. Tap the three vertical dots in the top-right corner.

2. Tap the "i" (Info) icon inside a circle at the very top of the menu pane.

3. Select Cookies & Site settings.

4. Tap the trash icon

5. Crucial Step: Swipe Chrome completely out of your phone’s App Switcher layout to kill the background process, then reopen the page.

6. Uninstall offline app

7. Reopen chrome and navigate to the apps url:
https://yiftachz.github.io/fdp-calc/

8. Tap Install on the new chrome app
